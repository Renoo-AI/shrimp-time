# Elite Asset Requirements (Total Project UX/UI Value: +$15M)

To fully realize the "Quiet Luxury" aesthetic and justify the elite status of this project, the following high-end assets are required:

## 1. Photography & Imagery
- **Style:** Chiaroscuro lighting, deep shadows, micro-contrast, macro shots of textures.
- **Subjects:** High-definition close-ups of pristine seafood (water droplets, ice crystals, rich textures of lobster shells).
- **Format:** Uncompressed 8K RAW files processed with subtle film grain. No stock-looking bright/even lighting.

## 2. Typography
- **Primary Display:** Licensed commercial versions of `Bodoni Moda` or similar ultra-high-contrast serif (e.g., *Ogg*, *GT Super*).
- **Secondary Body:** High-legibility geometric or neo-grotesque sans-serif with perfect kerning (e.g., *Jost*, *Aeonik*).

## 3. Motion & Interaction Assets
- **Video Backgrounds:** 4K 60fps slow-motion loops of dark ocean waves, ink diffusing in water, or subtle smoke/steam (used with mix-blend-mode).
- **3D Assets:** WebGL/Three.js optimized models of signature dishes that users can rotate with physics-based inertia.

## 4. UI Elements
- **Icons:** Custom-crafted SVG icon set with consistent $1	ext{px}$ stroke width, avoiding any generic libraries.
- **Textures:** Hand-crafted noise overlays and gradient meshes created specifically for this project's dark theme to prevent "banding" on high-end OLED displays.
